# [Microsoft Ignite 2025](https://ignite.microsoft.com)

## ðŸ”¥LAB510: The Power of GitHub Copilot in VS Code

### Session Description

Dive deep into the power of GitHub Copilot within VS Code. This hands-on lab will guide you through maximizing GitHub Copilot's potential for your daily coding tasks. Get firsthand experience at how Copilot can transform your workflow with agent mode, boost productivity without sacrificing code quality, model selection strategies and which models to leverage for your specific scenarios. Perfect for those looking to master GitHub Copilot in VS Code and drive impactful results within their teams.

### ðŸ§  Learning Outcomes

By the end of this session, learners will be able to:

- Use GitHub Copilot code completions to generate code from natural language
- Explore codebases using GitHub Copilot Chat with @workspace participant
- Provide project context through Copilot instructions files
- Use Agent mode to implement complex, multi-file changes autonomously
- Interact with Azure resources using natural language
- Query databases using natural language with GitHub Copilot extensions
- Generate Infrastructure as Code templates (Terraform or Bicep) with AI assistance
- Create CI/CD workflows using GitHub Copilot
- Understand tool-calling and Model Context Protocol (MCP) capabilities
